Plugin para la suscripción del sitio web, manda por correo el último post insertado a todos los suscriptores.

== Description ==

Este plugin permite que se suscriban por email por medio de un widget y cuando nosotros isertamos un nuevo post, ese lo manda por correo a todos los usuarios que se han suscrito, además tenemos el control de los emails y los podemos añadir o borrar. 

== Installation ==

Hay varias opciones para la instalación y configuración de este plugin.

= Carga Manualmente =

1. Descargar y descomprimir el plugin.
2. Subir carpeta 'suscriptflech' dentro del directorio '/wp-content/plugins/'.
3. Dirigase a la página de administración del plugin y activalo.

= Instalación vía área de Administración =

1. En el área de administración ir a Plugin > Add New y buscar 'suscriptflech'.
2. Click en instalar y luego en activar.

= Configura el Plugin =

1. entra en el área de administración de Wordress ir a setting > Suscriptores Admin y luego cambia el logo por el de tu empresa para que cuando les envie un email les salga su logo.


= Como usar el widget =

1. Ir a Appearance > Widgets y arrastra el Widget 'Formulario de suscripción' a la barra lateral principal.
2. Ingrese el titulo que desee.

== Frequently Asked Questions == 

= Como cambio el logo del correo que envío =

entra en el área de administración de Wordress ir a setting > Suscriptores Admin y luego cambia el logo por el de tu empresa para que cuando les envie un email les salga su logo.

== Screenshots == 

1. Una vez instalado el plugin, dirigase a Ajustes > Suscriptores Admin.
2. Seleccione cambiar imagen logo y inserte una.
3. Dale al boton guardar.
